The bend programs are linked so that linked program can be run after running the program. Cyclic link is possible when two programs are linked to each other. Bend program containing multiple setup sheet is linked by default.

To run linked program automatically in auto mode and without machine going to stop, after completing the current bend program, Connect program for same tool height should be turned ON in Machine Settings → General Parameters → Tools → Safety. Setup sheet is show if linked program has different tool height and machine will go to stop. To avoid this when Robot is connected, Connect all program in robot mode should be turned ON in Machine Settings → General Parameters → Tools → Safety.
