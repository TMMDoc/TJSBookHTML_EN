The Create page is used to create new press-brake programs. There are four program creation modes available, as seen from the four icons on this page:

From Profile: Create a program by drawing a new profile or reusing an existing profile and then extruding the profile to the desired length. The program is then automatically sequenced and tooled-up for bending. Profiles are stored in a list for later reuse.

Fold flat DXF: Create a program by starting with a flat DXF part (a development), and then fold it to create a 3D model. You can edit the bend lines in the DXF and set bend angles and directions before folding it up. The bends in this model are automatically sequenced and tooled-up for bending. Before folding up the flat pattern, make sure you enter the required parameters in the Settings dialog and verify the thickness and material used. If you plan to use special tools, you should select and lock them in the same dialog. In fact, those parameters influence the dimensions of the resulting folded 3D model. After this initial folding, the 3D model always maintains its dimensions even if you change the tooling using the Edit > Sequence > Recompute.

Import 3D model: Create a program by starting with a 3D model, an IGES or other common 3D formats (SolidWorks, SolidEdge, STEP, Parasolid etc). The bends in this model are then automatically sequenced and tooled up. When you import a MetaCAM 3D model, that is already tooled and sequenced, you will find that the model is re-tooled while importing it in RightAngle. To keep the tooling intact, you can go to the Edit page and use the import command that transfer provides. Once imported, you can either run or edit the program and then process the model on the brake.

Direct program: Create a program directly simply by entering back gauge positions, bending angles and bend lengths. The program can be run directly on the brake with a suitable blank.

In the first 3 methods, a 3D model of the part you are producing is available (or created) and can display 3D simulations of the bending process. It can also check that the part being bent will not collide with the press-brake or with the tooling. Fouling between different parts of the component during the bending cycle can also be checked. However, with the Direct Program method, these checks cannot be done since does not have a complete 3D model of the component you are trying to produce. Use this method with care.

From Profile

This option is used to create extrusions based on a profile which is defined in a parametric manner. When you press this option, displays the parametric profile shapes you have already created:

The buttons on the right are explained below:

New

You can sketch a completely new profile by touching New. The profile starts with a single straight section, but you can then add new sections and modify the lengths of sections and the angles between sections.

Edit

Select one of these profiles by touching it, and then touch Edit to modify it for use. From this, editor window, you can then set up a suitable material and thickness and extrude the profile to create a workpiece.

Clone / Delete

Use Clone to make a copy of the selected profile (so you can then modify the copy), or touch Delete to delete the selected profile.

Rename

Click on the Rename button and it opens a dialog (see picture below) to rename the selected profile.

Extrude

Select an existing profile and then click Extrude. The part settings dialog pops up where the value for Extrusion can be entered and then press the Done button to create it. See the section on Part Settings for more details.

Import

Import a dxf file which contains the side view of the part.

Editing the base segment length

Each profile being edited has a base segment; this is the horizontal segment near the center of the profile. The base segment of a profile is special; it acts like the anchor for the profile and you cannot change it to any angle other than horizontal. You can change the base segment’s length by touching one of its endpoints and dragging left or right; the figure below shows the locations where it is possible to touch and drag to change the base segment length for a profile:

Editing the length of any other segment

The length of any other segment can be edited by clicking on the free end of that segment and dragging the same. The free end of any segment is the end that is further away from the base.

As the segment is dragged, the length dimension of the segment gets updated continuously, so an immediate feedback on the actual length setting at the segment is achieved.

Changing a corner angle

To edit a corner angle, click on the free end of a segment and drag it sideways; this changes the angle at the start of the segment. The example below helps us understand how this works. As the segment is dragged to change the angle, the interior angle snaps to some common angles (like 45, 60, 90 and so on). In case of a non-standard angle, drag to set an approximate angle, and then tap on the angle display and type in the exact value.

The end of the segment closer to the base segment is fixed; the other end moves when you drag it.
You can change both the length and angle of a segment at the same time; dragging sideways changes the angle, while dragging along the segment changes the length.

Adding new segments to a profile

A new flange can be added to the profile at either of the free ends of the profile. To do this, click a small distance away from either of the free ends. If clicked too close to the free end, the segment also will be dragged along resulting in changing its length or angle. Touch about a centimetre or two away from the free end and start dragging to add a new segment to the profile:

Deleting a segment

A segment can be deleted in one of two ways: Tap on the segment length. It becomes editable. Type in 0 and press Enter; the segment gets deleted. Click on the free end of the segment and drag it toward the other end, shortening the segment. If it is dragged close enough, the segment disappears when you lift your finger (Without lifting, you can continue dragging and the segment will re-appear once you move a bit further away from the fixed end).

The base segment of a profile cannot be deleted.

Fold

Touching the Fold button will fold the DXF using the angles specified and displays a 3-dimensional view. This view can be rotated by touching on the blank area around the model and then dragging the view. Navigation buttons are present to the left side of the window. Use the arrow keys left, right, up and down to move the model along the respective direction. Touch the Plus mark to zoom-in and use the Minus mark to zoom-out the model. To fit the model to the window, touch the rectangle mark, present in between the Plus and Minus icons.

If this looks fine, then press the Do Tooling button to compute a bending sequence and tool assignments for this model. The model is then saved in the Models folder. Review the sequence, gauge positions, and tooling on this model, or run the program for this model on the press-brake. If the model does not look quite correct, you can tap the Cancel button to go back to the previous page, where you can adjust the folding angles and try again. Before folding up the flat pattern, make sure the required parameters are entered in the Settings dialog and the thickness and material used are verified. If you plan to use special tools, select and lock them in the same dialog. In fact, those parameters influence the dimensions of the resulting folded 3D model. After this initial folding, the 3D model always maintains its dimensions even if the tooling is changed using the Edit → Sequence → Recompute.

Import 3D Model

This option is used to import a 3D model from other format and use it for bending. The File Selection panel appears when this option is selected. Then pick a 3D file to import.
The following formats are supported:

  1. IGES 3D model (. iges, .igs files)

After the file is selected, the 3D model data is imported and displayed. Use the screen to rotate the model around and verify that the import looks correct. In the displayed model, bend lines are marked in yellow, while cut/weld edges are marked in black:

If the model looks OK, use the Settings button to set up the material, thickness for the model, as well as the punch and die to be used.

Use the unfold button to view the flat pattern and the flat dimensions of the imported file in a new window. Upon pressing the back button, the flat pattern window closes, and returns to the 3D Import window.

Touching the Do Tooling button process the model by computing a bend sequence, back-gauge positions and the tooling setup.

Direct Program

The Direct Program environment is used to create a simple bending program directly on the control. The program is created by specifying the bend length, the bend angle and the gauges positions for each bend. If values outside expected ranges are entered, the input is refused, and the expected range is displayed (in the top right corner). The error message is transient - it disappears after a few seconds.

When this program is run, the setup screen displays all the punches and dies used, along with their 'flip' states (so it can be mounted correctly by the operator):

Settings

Touch the Settings button on the top left to bring up the Part Settings dialog where you can configure the settings like material, thickness, punch, die etc.

Add / Remove

Touch the Add button on the bottom left panel to add bends into the program and touch the Remove button to remove them.

Move up/ Move down or Prev Bend / Next Bend

The Move up and Move down buttons are used to shuffle the currently selected bend up or down in the sequence. The currently selected bend is displayed in bold color, with the previous and next bends also displayed in a shadowed color. You can touch the Prev Bend or Next Bend button to display other bends for editing.

Gauging

The currently selected bend is displayed in detail on the right panel, and various settings shown can be edited. To understand the meanings of Gauge Z1, Gauge Z2, Gauge R1 and Gauge R2, its necessary to understand the coordinate system used by:

  1. The Z1 and Z2 axes are the gauge positions along the machine table.

  2. X axes are gauge distances from the bend line, with increasing X going into the machine.

  3. The R axis is the vertical positions of the gauges, with positive R going upwards.

When the outer height of the flange that needs to be bent is specified, and the gauge position is computed automatically. This computation takes the material, thickness and die opening into account (you can toggle between the two to see the actual Gauge-X that has been computed for a given flange height). On machines where the two gauges can move independently in X, you can input the values for X1 and X2 using the input boxes provided.

Jog

Touch the Jog button to bring up the Axis Jog Panel. This allows the user to freely move all the gauge axes using either the handwheel, or by holding down the jog buttons. From this panel, one can record the position of the gauges directly into the program by tapping the teach button there.

Table Programming

Touch the table to allow editing the bending program in tabular form. This is disabled by default. To enable it, turn ON Allow table programming in Settings → Other Settings → Machine Settings → Display. See Table Programming for details.

Run

Touch the Run to run the currently editing part. If the tooling of the part doesn’t differ from the last run setup, setup sheet will be skipped.
